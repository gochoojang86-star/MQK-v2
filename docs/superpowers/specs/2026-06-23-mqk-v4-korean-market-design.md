# MQK v4 — 국장 세력주 스나이퍼 설계 스펙

**작성일**: 2026-06-23  
**상태**: 설계 확정, 구현 대기  
**배경**: v3(미네르비니 SEPA 기반)가 국장 구조와 맞지 않아 전면 재설계. 세력주 매매 + 테마 선도주 단기 스윙 전략으로 교체.

---

## 1. 전략 철학 변경

### v3 vs v4 핵심 차이

| 항목 | v3 (미네르비니) | v4 (국장 세력주) |
|---|---|---|
| 종목 발굴 | VCP/박스돌파 패턴 탐색 | 상한가 + 거래대금 폭증 포착 |
| 진입 기준 | EP 갭상승 첫 눌림, 강한 촉매 | 세력 확인 후 눌림 (장전 + 장중) |
| 홀딩 | 추세 지속 기반 (주 단위) | 세력 신호 유지 시 (2~5일) |
| 청산 | 기계적 손절 + 이동평균 이탈 | 신호 기반 (거래대금 급감, 수급 전환, 테마 소멸) |
| LLM 역할 | 발굴 + 진입 + 홀딩 전 판단 | 세력 검증 + 신호 해석 + 청산 판단 |

### 왜 v3가 국장에 맞지 않는가

- **미네르비니 VCP**: 기관이 8~12주에 걸쳐 물량을 쌓는 패턴 전제 → 국장 세력은 1~2주 안에 완성
- **EP 갭상승**: 미장은 수개월 추세의 출발점, 국장은 1~3일 안에 되돌림
- **홀딩 기간**: 미장 기준(주~개월) vs 국장 테마 사이클(2~5일)
- **결과**: 살 만한 종목 못 찾고, 찾아도 기준 미달로 HOLD

## 2. 셋업 타입 (진입)

### LIMIT_UP_PULLBACK — 상한가 눌림
- **조건**: 전일 상한가 → 다음날 시가 갭다운 -3% 이내 → 장초반 -3~5% 눌림
- **장전 확인**: 07:30~08:30 시간외 거래에서 매도 물량 없는지 사전 검증
- **LLM 판단**: 거래대금 유지 여부, 세력 이탈 징후 없는지
- **목표 홀딩**: 2~3일

### VOLUME_SURGE_LEADER — 거래대금 폭증 선도주
- **조건**: 거래대금 전일 대비 5배↑ + 당일 10%↑ 강한 양봉 + 테마 내 거래대금 1위
- **LLM 판단**: 세력 vs 개미 구분 (거래대금 연속성, 뉴스 촉매 강도)
- **진입**: 당일 첫 눌림
- **목표 홀딩**: 1~3일

### THEME_CATALYST — 테마 뉴스 촉매
- **조건**: 정책 발표/대형 수주/산업 구조 변화 등 강한 촉매 뉴스 + 테마 내 거래대금 1위 + 당일 5%↑
- **LLM 판단**: 촉매 강도 (단순 기사 vs 구조적 변화), 테마 지속성
- **진입**: 당일 눌림 or 다음날 갭업 보합 시
- **목표 홀딩**: 2~4일

## 3. 청산 신호 (출구)

손절도 기계적 -3%가 아닌 **신호 기반**. LLM이 신호를 해석해서 청산을 결정한다.

| 신호 | 조건 | 청산 타이밍 |
|---|---|---|
| **VOLUME_DRY** | 거래대금 전일 대비 -50% 급감 | 다음날 시가 청산 |
| **FLOW_REVERSAL** | 기관 + 외인 동시 순매도 2일 연속 | 당일 청산 |
| **THEME_FADE** | 테마 뉴스 소멸 + 섹터 거래대금 감소 | 다음날 시가 청산 |
| **PRICE_SIGNAL** | 세력 지지선 하향 돌파 + 거래대금 동반 증가 | 즉시 청산 (신호 손절) |
| **LIMIT_UP_FAIL** | 장중 상한가 근접 후 밀리면서 거래대금 폭발 | 즉시 청산 |

**PRICE_SIGNAL 핵심 원칙**: 거래대금 없이 그냥 밀리는 건 손절 안 한다. 거래대금이 동반될 때만 세력 매도로 판단.

---

## 4. 세력 판별 신뢰도 기준

LLM이 세력 vs 개미를 구분할 때 사용하는 신호 (신뢰도 순):

1. **거래대금 연속성** (최고) — 2~3일 연속 유지 = 진짜 세력. 하루 터지고 끝 = 개미 or 뉴스 반응
2. **상한가 다음날 시가 흐름** (높음) — 갭업/보합 유지 = 세력 의지, 갭다운 -3%↑ = 세력 이탈
3. **기관·외인 참여** (중간) — 있으면 더 신뢰. 없어도 진입 가능. 파는 건 위험 신호
4. **호가창 잔량** (낮음) — 허수 가능성으로 보조 참고만

```
코드:  전일 상한가 종목 리스트 추출 (get_limit_up_stocks)
```
